# 概要

indexedDB用ドライバ。

| 関数           | 内容     |
| -------------- | -------- |
| initDB         | DB初期化 |
| deleteDB       | DB削除   |
| getData        | 条件取得 |
| getData_all    | 一括取得 |
| addData        | 追加     |
| setData_all    | 一括更新 |
| setData        | 条件更新 |
| deleteData_all | 一括削除 |
| deleteData     | 条件削除 |
